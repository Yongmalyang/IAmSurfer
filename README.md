* 조이스틱으로 조작하는 점수 갱신형 게임
* 망망대해에서 방해물들을 피해 서핑을 즐겨보자
* 서핑 중 만나는 블루홀은 깊고 위험하니 가까이 가지 말자
* 블루홀에 빠졌다면, 제한 시간 안에 서핑보드가 있는 수면으로 올라오자
